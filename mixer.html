Aux Sends & Buses

Parallel Effects Processing

Professional workflow using aux sends for reverb and delay:

  • ‚úì Create Bus: Add reverb/delay bus in project
  • ‚úì 100% Wet: Bus effect set to full wet (no dry signal)
  • ‚úì Add Sends: Each track has send controls
  • ‚úì Send Level: Control how much signal goes to bus
  • ‚úì Blend: Mix dry track + wet bus return
  • ‚úì Efficiency: One reverb for multiple tracks
  • ‚úì Consistency: Same reverb space for cohesion
  • ‚úì CPU Savings: Much more efficient than insert reverb per track

  • ‚úì Color Coded: Green (safe), Yellow (hot), Red (clipping)

Professional Mixing Workflow

  • ‚Ä¢ Set Levels First: Get rough balance with faders before adding effects
  • ‚Ä¢ Pan for Space: Spread instruments across stereo field (drums center, guitars left/right)
  • ‚Ä¢ EQ Subtractive: Cut problem frequencies before boosting
  • ‚Ä¢ Reverb on Sends: Use aux sends for reverb/delay, not inserts
  • ‚Ä¢ Compression Gentle: Start with 2:1 or 3:1 ratio, increase if needed
  • ‚Ä¢ Monitor Meters: Keep peaks below -6dB to -3dB for headroom
  • ‚Ä¢ Master Limiting: Add limiter on master to prevent clipping
  • ‚Ä¢ A/B Reference: Compare your mix to professional tracks in same genre
  • ‚Ä¢ Take Breaks: Ear fatigue is real - rest every 30-45 minutes
